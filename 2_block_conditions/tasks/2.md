# Задача «Модуль разности двух чисел»

Даны два целых числа. Вывести на печать модуль разности этих чисел.

Напомним,

> |a| = a, если a >= 0;
>
>|a| = -a, если a < 0.

**Входные данные:**

Два целых числа.

**Выходные данные:**

Модуль разности двух чисел.

**_Пример 1_**

Входные данные:

```
10
8
```

Результат:

```
2
```

> **Пояснение:**
>
> |10 - 8| = |2| = 2.



**_Пример 2_**

Входные данные:

```
7
12
```

Результат:

```
5
```

> **Пояснение:**
>
>|7 - 12| = |-5| = 5.
