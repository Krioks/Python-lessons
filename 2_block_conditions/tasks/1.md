# Задача «Минимум из двух чисел»

Даны два целых числа. Выведите значение наименьшего из них.

**Входные данные:**

Два целых числа.

**Выходные данные:**

Наибольшее число.

**_Пример 1_**

Входные данные:

```
45
-1
```

Результат:

```
-1
```

**_Пример 2_**

Входные данные:

```
5
14
```

Результат:

```
5
```

**_Пример 3_**

Входные данные:

```
77
77
```

Результат:

```
77
```
