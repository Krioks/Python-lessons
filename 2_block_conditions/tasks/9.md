# Задача «Определение четверти»

Даны целые (не равные нулю) значения координат (x; y) точки. Определите четверть, в которой лежит точка.

**Входные данные:**

Координаты точки.

**Выходные данные:**

Номер четверти.

**_Пример 1_**

Входные данные:

```
2
3
```

Результат:

```
1
```

> **Пояснение:**
>
> Точка с координатами (2; 3) лежит в 1-й четверти.


**_Пример 2_**

Входные данные:

```
-4
5
```

Результат:

```
2
```
