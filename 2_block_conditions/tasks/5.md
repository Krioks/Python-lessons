# Задача «Решение уравнения ax+b = 0»

Даны целые числа **a** и **b**. Напишите программу, которая решает уравнение вида a*x + b = 0. Результат вывести на
печать.

**Входные данные:**

Два целых числа.

**Выходные данные:**

Решение уравнения.

**_Пример 1_**

Входные данные:

```
2
4
```

Результат:

```
-2
```

**_Пример 2_**

Входные данные:

```
0
0
```

Результат:

```
Корень - любое число
```

**_Пример 3_**

Входные данные:

```
0
5
```

Результат:

```
Корней нет
```
