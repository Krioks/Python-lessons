# Задача «Следующее четное»

Дано целое положительное число. Найдите следующее за ним четное число.

> **Запрещено использовать условный оператор `if`!**

**Входные данные**

Вводится число.

**Выходные данные**

Выведите ответ на задачу.

**_Пример 1_**

Входные данные:

```
8
```

Выходные данные:

```
10
```

**_Пример 2_**

Входные данные:

```
7
```

Выходные данные:

```
8
```
