число записано в отдельной строке.

**Входные данные**

В двух строках вводятся два числа (числа целые,положительные, не превышают 1000).

**Выходные данные**

Выведите ответ на задачу.

**_Пример 1_**

Входные данные:

```python
leg1 = 3
leg2 = 4
```
Выходные данные:

```
Площадь треугольника = 6
```
